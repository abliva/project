# 个人健康管理系统

## 项目简介

个人健康管理系统是一款基于 Python 开发的综合性健康管理桌面应用程序，旨在帮助用户全面记录、分析和追踪个人及家庭成员的健康数据。系统采用现代化的图形用户界面（GUI），提供直观的数据可视化和智能健康分析功能。

## 核心功能

### 1. 用户管理系统
- **多用户支持**：支持管理员、普通用户、家属用户三种角色
- **安全认证**：采用 SHA-256 加密算法保护用户密码
- **家属关系管理**：支持建立家庭成员间的关联关系，实现数据共享与权限控制
- **个人信息管理**：记录用户的姓名、性别、出生日期、联系方式等基本信息

### 2. 健康数据记录
系统支持记录以下健康指标：

| 健康指标 | 数据类型 | 单位 | 正常范围参考 |
|---------|---------|------|-------------|
| 身高 | 浮点数 | 米 (m) | - |
| 体重 | 浮点数 | 千克 (kg) | 根据身高计算 BMI |
| 血压 | 字符串 | mmHg | 收缩压: 90-140, 舒张压: 60-90 |
| 血糖 | 浮点数 | mmol/L | 空腹: 3.9-6.1, 餐后2h: 4.4-7.8 |
| 心率 | 整数 | 次/分钟 | 60-100 |
| 备注 | 文本 | - | - |

### 3. 智能健康分析

#### BMI 智能分析
- 自动计算身体质量指数 (BMI)
- 根据中国成人标准进行分类：
  - 偏瘦: BMI < 18.5
  - 正常: 18.5 ≤ BMI < 24
  - 超重: 24 ≤ BMI < 28
  - 肥胖: BMI ≥ 28
- 使用颜色编码直观显示健康状态

#### 血压分类系统
- 低血压: 收缩压 < 90 或 舒张压 < 60
- 正常: 收缩压 < 120 且 舒张压 < 80
- 正常偏高: 收缩压 < 130 且 舒张压 < 80
- 高血压1级: 收缩压 < 140 或 舒张压 < 90
- 高血压2级: 收缩压 < 180 或 舒张压 < 120
- 高血压危象: 更高值

#### 血糖评估
- 低血糖: < 3.9 mmol/L
- 正常: 3.9-6.1 mmol/L (空腹)
- 糖尿病前期: 6.1-7.0 mmol/L
- 糖尿病: > 7.0 mmol/L

#### 心率监测
- 心动过缓: < 60 次/分钟
- 正常: 60-100 次/分钟
- 心动过速: > 100 次/分钟

### 4. 数据可视化
- **趋势图表**：使用 Matplotlib 绘制健康数据的时间序列变化趋势
- **多维度展示**：支持按日期范围筛选数据
- **交互式界面**：图形化界面嵌入 Tkinter 应用程序中
- **中文完美支持**：配置中文字体，确保标签和标题正确显示

### 5. 提醒与报警系统
- **健康提醒**：支持设置用药提醒、复诊提醒、运动提醒等多种类型
- **灵活时间设置**：可自定义提醒时间和重复周期（每周特定天数）
- **异常报警**：当检测到健康指标超出正常范围时自动生成报警记录
- **通知机制**：支持邮件通知功能（SMTP）

### 6. 数据管理
- **SQLite 数据库**：轻量级本地数据库存储
- **数据持久化**：所有健康记录安全保存
- **标准参考值维护**：内置各年龄段、性别的健康指标正常范围
- **数据导出**：支持导出为 CSV 格式便于进一步分析

## 技术架构

### 技术栈
```
前端界面: Tkinter + ttk (Python GUI)
数据库: SQLite3
数据可视化: Matplotlib + NumPy + Pandas
数据处理: Python 标准库 (datetime, hashlib, csv, json)
邮件服务: smtplib + email
加密算法: SHA-256
线程处理: threading
```


### 数据库设计

系统使用 SQLite 数据库，包含以下核心表：

1. **users** - 用户信息表
   - 存储用户基本信息、角色权限、登录凭证
   
2. **family_relationships** - 家属关系表
   - 定义家庭成员间的关联和访问权限
   
3. **health_records** - 健康记录表
   - 存储每日健康指标数据
   
4. **health_reminders** - 健康提醒表
   - 管理各类健康提醒设置
   
5. **health_alerts** - 报警记录表
   - 记录异常健康指标的报警信息
   
6. **health_standards** - 标准参考值表
   - 维护各指标的正常范围标准
   
7. **user_settings** - 用户设置表
   - 存储个性化配置选项

## 使用指南

### 1. 登录系统
启动程序后，使用默认管理员账户或创建新账户登录。

### 2. 记录健康数据
1. 选择"添加记录"功能
2. 输入当日的各项健康指标
3. 系统会自动计算 BMI 并进行健康评估
4. 如有异常指标，系统将自动报警

### 3. 查看数据分析
1. 进入"数据统计"模块
2. 选择要查看的指标类型和时间范围
3. 查看趋势图表和分析报告

### 4. 设置健康提醒
1. 打开"提醒设置"页面
2. 创建新的提醒任务
3. 设置提醒时间和重复规则
4. 可选择启用邮件通知

### 5. 家属管理（管理员功能）
1. 创建家属账户
2. 建立家属关系
3. 配置查看/编辑权限
4. 实现家庭健康数据共享
